## рџљЂ Getting Started

### 1. Install dependencies

```bash
cd AI_Military_Intelligence
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Train the machine-learning models

```bash
python train_models.py
```

This reads `data/globalterrorism.csv` and saves two trained models into
`models/`:

- **Attack Type Prediction** вЂ” predicts `attacktype1_txt` from country,
  region, weapon, target, group, and incident outcome fields.
- **Threat Level Prediction** вЂ” predicts LOW / MEDIUM / HIGH severity from
  pre-attack context (country, region, attack type, weapon, target type,
  suicide flag).

This step only needs to be run once, or again whenever the dataset changes.

### 3. Launch the app

```bash
streamlit run Home.py
```

The app opens automatically in your browser (default:
`http://localhost:8501`).

## рџ§  Machine Learning Details

Both models are `RandomForestClassifier`s from scikit-learn, trained with
categorical feature encoding via `LabelEncoder`. Model artifacts
(`.pkl` files), encoders, and feature importances are saved with `joblib`
and loaded once per session using Streamlit's `st.cache_resource`, so
predictions stay fast without retraining on every page interaction.

## рџ“Љ Dataset

This project uses the **Global Terrorism Database (GTD)**, an open-source
database of terrorist incidents maintained by the National Consortium for
the Study of Terrorism and Responses to Terrorism (START) at the
University of Maryland. Place your GTD CSV export at
`data/globalterrorism.csv` (UTF-8/Latin-1 encoded) to use the dashboard
with your own copy of the data.
